# Machine Learning Experiments for Researchers

Practical companion to the [IMT Machine Learning course](https://cse.lab.imtlucca.it/~bemporad/ml.html).


The course covers key techniques and best practices for designing, running, 
and analyzing machine learning experiments.


By the end, you should have:
- A minimal, working training loop you can run anywhere (Colab or local)
- A research-grade training script (configs, logging, checkpoints) you can trust for comparisons
- (Optional) A tiny text experiment (mini Transformer) + hardware intuition for scoping projects
